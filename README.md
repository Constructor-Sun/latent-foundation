# Latent Foundation

This repository documents my personal exploration of foundational latent representation learning. It serves as a codebase and mathematical foundation for furture exploration.

## Currently Included

- Variational Autoencoders (VAE)
- Vector Quantized VAE (VQ-VAE)
- Diffusion Models
- CLIP-style multimodal representation learning
- Flow Models (waiting to be implemented...)
